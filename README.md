- Engaging Quiz Interface: Clean and intuitive UI for a seamless quiz experience.
- Question Progress Tracking: A ProgressBar and a text counter (1/10) keep users informed of their progress.
- Interactive Options: Users can select an answer from four options, with clear visual feedback.
- Immediate Answer Validation: Options are highlighted in green (correct) or red (incorrect) after checking the answer.
- Dynamic Question Loading: The app cycles through a predefined list of questions, each with a unique flag.
- Score Calculation: The app keeps track of the user's score as they progress through the quiz.
- Final Result Screen: At the end of the quiz, users are navigated to a results page showing their final score and name, from where they can go to the leaderboard page.
- Firebase Realtime Database: Utilized to store user data and power a live leaderboard.
